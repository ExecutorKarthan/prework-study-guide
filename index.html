Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is a framework in which we write web-related applications
  • HTML is how we organize text and layout but is not logical code that completes tasks

CSS

  • CSS provides formatting to the HMTL! It allows for num erous types of self expression
  • CSS files are modular, similar to an import from python or a java class file
  • CSS files are settings based, but must be linked to their affected HTML pages by adding a link in the page header.

Git

  • Git is a version contrl software
  • Git can be done via commandline or by its desktop app
  • Git changes are PUSHED to the web repository
  • Git changes are PULLed into a main branch via pull requests

JavaScript
